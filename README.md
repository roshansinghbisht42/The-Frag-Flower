# 🌸 The Frag Flower

A modern and visually appealing flower showcase website built using **HTML, CSS, JavaScript, and Firebase Authentication**.  
This project demonstrates frontend development skills along with user authentication and UI/UX design.

---

## 🚀 Features

✨ **User Authentication (Firebase)**
- Secure login & signup system
- Email verification functionality
- Session handling using localStorage

🎨 **Beautiful UI/UX**
- Responsive design for multiple screen sizes
- Smooth animations and transitions
- Attractive gradient themes

🌙 **Dark Mode Toggle**
- Light/Dark theme switch
- User preference saved in localStorage

🖼️ **Flower Gallery**
- Interactive gallery of rare flowers
- External links to Wikipedia for learning
- Hover effects with animations

🔐 **Logout System**
- Clears session data
- Redirects to login page securely

⚡ **Smooth Scrolling**
- Implemented using Lenis library for better user experience

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & Animations
- **JavaScript (ES6)** – Logic & Interactivity
- **Firebase Authentication** – User login/signup system
- **Lenis Library** – Smooth scrolling
