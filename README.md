# 🌸 Frag Flower – Fragrance Website

Frag Flower is a responsive web application designed to showcase and explore fragrance products. It includes user authentication and email verification using Firebase, providing a secure and interactive user experience.

---

## 🚀 Features

- 🔐 User Authentication (Login & Signup)
- 📧 Email Verification using Firebase
- 🌐 Responsive UI (Mobile + Desktop)
- 🛍️ Fragrance product browsing interface
- ⚡ Fast and interactive user experience

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend/Services:** Firebase Authentication  
- **Database:** Firebase
